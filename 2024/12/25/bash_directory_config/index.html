在 Bash 中隐藏路径以简化提示符

在 Bash 中，可以通过以下方法隐藏路径或调整提示符以简化显示，并保持高亮效果：

方法一：自定义 PS1（Prompt String 1）

PS1 是 Bash 的提示符变量，你可以通过设置它来定制显示内容。例如：

  • \u：显示用户名。
  • \h：显示主机名。
  • ~：始终显示为 ~，而不是完整路径。

如果希望添加高亮效果，可以加入颜色代码：

  • \[\033[1;32m\]：设置绿色高亮（用户名和主机名）。
  • \[\033[1;34m\]：设置蓝色高亮（路径）。
  • \[\033[0m\]：重置颜色，避免影响后续输出。
