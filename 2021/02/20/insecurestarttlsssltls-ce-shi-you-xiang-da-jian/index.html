Insecure/STARTTLS/SSLTLS 测试邮箱搭建

发表于 2021-02-20 更新于 2021-06-28

安装Hmailserver(https://www.hmailserver.com/), 需要提前安装.netformwork；

按照提示设置密码；

使用hMailServer Administrator连接到hmailserver, 密码是安装的时候设置的；

首先添加一个domain；

接着添加几个账号，设置好密码；

生成测试证书，可以去https://myssl.com/create_test_cert.html

可以看到以下三个文件：

cert.pem对应服务器证书
full_chain.pem对应CA证书
private.key对应私钥

回到hMailServer Administrator, 打开Settings->Advanced->SSL certificates；

点Add 新加一个证书；

给证书起一个名字，Certificate file 对应cert.pem，Private key file对应private.key；

然后保存即可；

切换到Settings->Advanced->TCP ports;

这里可以按照需要进行设置相应的端口:

一般对应如下:
SMTP: 25(Insecure)/587(STARTTLS)/465(SSLTLS).
POP3: 110
IMAP:143

对于测试部分，这里只是用到了SMTP的发送功能，收件功能没有特殊要求。

选中0.0.0.0/587/SMTP, Connection security 选择 STARTTLS(Required), SSL Certificate选择步骤10里面的证书， 然后保存，这样就配置好了STARTTLS;

同样，选中0.0.0.0/465/SMTP(如果没有参考PORT 25来配置一个就好)，Connection security 选择SSL/TLS, SSL Certificate选择步骤10里面的证书然后保存，这样就配置好了SSL/TLS；

到此，邮件服务器就已经配置完成；

使用邮件客户端进行测试，可以用thunderbird 或者foxmail;

分别指定不同的SMTP发送服务器设置；

确保可以收到邮件；

检查hMailSever的log，可以看到对应的加密log:
